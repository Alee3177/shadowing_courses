「から」句型｜跟讀練習

請先按「解鎖／播放本句」
句型：原因 ＋ から、結果。／ 因為……，所以……。

同動詞問答練習

跟讀方式

1. 看日文
先看第一欄日文，聽一次。
2. 看平假名 / 羅馬拼音
確認發音，再跟讀一次。
3. 看中文
理解意思：因為……所以……。
4. 問答練習
用「どうして＋動詞＋か」問原因。

問答句型：以動詞為主

日文平假名 / 羅馬拼音中文
どうして掃除しますか。
汚いですから、掃除します。		どうしてそうじしますか。 / doushite souji shimasu ka.
きたないですから、そうじします。 / kitanai desu kara, souji shimasu.		為什麼要打掃？
因為很髒，所以打掃。
どうして薬を飲みますか。
頭が痛いですから、薬を飲みます。		どうしてくすりをのみますか。 / doushite kusuri o nomimasu ka.
あたまがいたいですから、くすりをのみます。 / atama ga itai desu kara, kusuri o nomimasu.		為什麼吃藥？
因為頭痛，所以吃藥。
どうして何も買いませんか。
お金がありませんから、何も買いません。		どうしてなにもかいませんか。 / doushite nani mo kaimasen ka.
おかねがありませんから、なにもかいません。 / okane ga arimasen kara, nani mo kaimasen.		為什麼什麼都不買？
因為沒有錢，所以什麼都不買。

可替換時態應用

日文平假名 / 羅馬拼音中文
汚いですから、掃除します。きたないですから、そうじします。 / kitanai desu kara, souji shimasu.因為很髒，所以打掃／會打掃。
汚かったですから、掃除しました。きたなかったですから、そうじしました。 / kitanakatta desu kara, souji shimashita.因為之前很髒，所以打掃了。
汚くありませんから、掃除しません。きたなくありませんから、そうじしません。 / kitanaku arimasen kara, souji shimasen.因為不髒，所以不打掃。
汚くありませんでしたから、掃除しませんでした。きたなくありませんでしたから、そうじしませんでした。 / kitanaku arimasen deshita kara, souji shimasen deshita.因為當時不髒，所以沒有打掃。
