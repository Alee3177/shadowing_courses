# ndesuga_sakura_shadowing

GitHub Pages upload structure:

```
shadowing_courses/lessons/ndesuga-sakura/
├── index.html
├── ndesuga_stpmpov_shadowing.html
├── data/ndesuga_stpmpov_shadowing.json
├── audio/n001.mp3 ... n022.mp3
└── tools/generate_audio.py
```

Note: bundled MP3 files are generated from romaji by local espeak as an offline fallback. For more natural Japanese audio, regenerate MP3 files with your preferred Japanese TTS and keep the same filenames.
